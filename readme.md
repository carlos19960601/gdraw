#  gdraw 用几何图形画图

非原创，主要用来学习，原始仓库：[https://github.com/fogleman/primitive](https://github.com/fogleman/primitive)

# Example

原始图

![](./examples/beauty.jpg)

```
./gdraw -i beauty.jpg -n 1000  -o ./beauty.out.png
```
